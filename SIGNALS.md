## Формат карточки-сигнала

Каждый рыночный/нарративный материал оформляется как запись в массиве `SIGNALS` (в `index.html`).

| Поле | Назначение |
|------|-----------|
| `date` | Дата (YYYY-MM-DD) |
| `cat` / `catLabel` | Категория: macro/narrative/onchain/mining/layer2 + эмодзи |
| `dir` | Направление для BTC: `neg` 🔴 / `pos` 🟢 / `neu` ⚪ |
| `signal` | Суть одной фразой |
| `data[]` | Ключевые цифры (чипы) |
| `context` | Предыстория, аналогии |
| `caveat` | Риски, оговорки, чего не знаем |
| `source` | Источник |

> Сводка 🔴/🟢/⚪ и фильтры строятся автоматически из данных.
> Добавить сигнал = добавить объект в массив `SIGNALS`.

- **Что это:** Первый месяц в истории, когда ежемесячный объём Lightning превысил $1 млрд (ноябрь 2025). Средний размер транзакции почти удвоился — с $118 до $223 г/г.
- **Контекст:** Lightning запущена в 2018 для микроплатежей. Рост среднего чека сигнализирует, что сеть перешла к реальным бизнес-расчётам: биржи переводят через неё живые деньги.

- **Оговорки:** Объём Lightning сложно верифицировать — приватные каналы не видны. Рост среднего чека может означать концентрацию среди крупных игроков, а не массовое принятие.
