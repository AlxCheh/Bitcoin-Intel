## Навигация — 3 кластера

Двухуровневая система: нижняя панель кластеров (LIVE / KNOWLEDGE / CONTEXT) + подвкладки сверху + строка-крошки «КЛАСТЕР / Вкладка».

| Кластер | ID | Вкладки сейчас | В очереди |
|---------|----|----------------|-----------|
| **LIVE** | live | ОБЗОР, АНАЛИТИКА, ПУЛЫ, РЫНОК | МЕМПУЛ |
| **KNOWLEDGE** | knowledge | ОСНОВЫ, ТЕХНОЛОГИИ | БЕЗОПАСНОСТЬ |
| **CONTEXT** | context | ИСТОРИЯ | — |

**Критерий:** LIVE = меняется в реальном времени, KNOWLEDGE = стабильная теория, CONTEXT = прошлое.

**Очередь наполнения** (только после материала от пользователя): МЕМПУЛ → БЕЗОПАСНОСТЬ

## Дизайн-система

**Стиль:** Arkham Intelligence — терминальная эстетика, строгие линии, без скруглений.

```
Фон основной:   #06080B
Фон карточки:   #0C0F14
Фон вложенный:  #111520
Линии/границы:  #1A1F2A, #242A38

Акцент BTC:     #F7931A  ← сохранять всегда
Текст основной: #D4DCE8
Текст dim:      #4A5568
Зелёный:        #22C55E
Красный:        #EF4444

Шрифты:
  IBM Plex Mono — метки, числа, навигация
  IBM Plex Sans — основной текст
```

### Правила дизайна
- Не менять цветовую схему без явного запроса
- Не удалять существующие разделы и контент
- Акцент #F7931A — обязательный, только Bitcoin
- Только Bitcoin — никаких других криптовалют
