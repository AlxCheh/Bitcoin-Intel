# Sprint 0 — организационные шаблоны (черновики)

> **Статус: ЧЕРНОВИКИ, не финальные документы.** Каждый файл в этой папке —
> структура и placeholder-заполнение, а не готовое решение. Реальные имена,
> каналы, роли и критерии подставляет команда — я (Claude) не располагаю
> этой информацией и не выдумываю правдоподобные, но фиктивные ответы вместо
> честного плейсхолдера.

## Зачем отдельная папка, а не сразу в целевые файлы

Каждый шаблон целится в конкретный конечный документ (`CONTRIBUTING.md`,
`docs/ONBOARDING.md`, новый `docs/RACI.md`) — но помещать незаполненный
черновик прямо в целевой файл означало бы либо ломать существующую
структуру документа наполовину готовым разделом, либо создавать
видимость, что организационная часть закрыта, когда на самом деле она
ждёт решения. Здесь — до момента, когда есть реальные ответы для переноса.
